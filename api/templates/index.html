Manual Playlist Data Generator

What is this?

This is a seperate too used to circumvent large Redis operations as well as circumvent Vercel serverless runtime limits.
Some songs and images are quite large and as such reach the limitations for the free tier of Vercel's services.
To circumvent this, you can run this tool locally to pre-generate the json file containing the playlist data and upload it to any file hosting service.

Usage

You can use a service such as Pastebin to host your .txt file.
Set it as temporary as a new json file containing the actual playlist data will be generated and return to you
(this txt file is only used to generate this json file)

For example your .txt file could look like this:

https://someurl.com/song1.flac
https://someurl.com/song2.flac
https://someurl.com/song3.flac

Enter a direct link to a .txt file containing a list of direct links to FLAC Vorbis or MP3 files.
